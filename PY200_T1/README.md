# Mean Median Mode (First Principle)

This project implements Mean, Median, and Mode using basic Python constructs.

## Rules Followed:
- No inbuilt functions like sum(), sort(), etc.
- Only loops and conditions used

## Concepts Used:
- Loops (for)
- Conditional statements (if-else)
- Manual sorting (Bubble Sort)

## Output:
Mean, Median, and Mode of a given list.
